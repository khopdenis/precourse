# gpu-sim-portfolio

Учебно-портфолийный репозиторий пути «бэкенд → GPU-симуляция жидкостей и разрушений».
Стартовая заготовка для подготовительного курса (тикеты PREP-*) и далее — основного
36-недельного плана.

## Сборка и запуск (одна команда конфигурации + одна сборки)

Linux / macOS:
```bash
cmake -S . -B build
cmake --build build
./build/hello
```

Windows (x64 Native Tools Command Prompt или PowerShell с установленной Visual Studio):
```bat
cmake -S . -B build
cmake --build build --config Debug
build\Debug\hello.exe
```

Ожидаемый вывод: `Hello, world! gpu-sim-portfolio builds fine.`

## Как устроен репозиторий

```
CMakeLists.txt        — вся конфигурация сборки (одна на проект)
src/main.cpp          — главная цель `hello` (тикет PREP-1.1)
exercises/*.cpp       — КАЖДЫЙ файл здесь автоматически становится
                        отдельным исполняемым файлом с именем файла.
                        Пример: создаёшь exercises/p1_3_swap.cpp →
                        появляется цель p1_3_swap → ./build/p1_3_swap
BACKLOG.md            — бэклог курса (добавь сюда файл из тикета PREP-1.2)
```

Механизм exercises сделан специально под подготовительный курс: там много
маленьких самостоятельных программ (swap, const, утечка, бенчмарки), и каждая
живёт в своём файле без правки CMakeLists.

После добавления нового .cpp в exercises/ перезапусти конфигурацию:
`cmake -S . -B build && cmake --build build` (флаг CONFIGURE_DEPENDS обычно
подхватывает новые файлы сам, но пересоздание конфигурации — надёжный путь).

## Санитайзер памяти (понадобится в занятии П2, тикет PREP-2.2)

GCC/Clang (Linux, macOS, WSL):
```bash
cmake -S . -B build-asan -DSANITIZE=ON
cmake --build build-asan
./build-asan/<цель>
```
При утечке или ошибке памяти программа напечатает подробный отчёт со строкой кода.
На чистом MSVC флаг игнорируется (у Visual Studio свой ASan — включается в свойствах
проекта; либо используй WSL).

## Предупреждения компилятора

Включены всегда (`-Wall -Wextra` / `/W4`). Правило курса (Definition of Done):
код собирается **без warnings**. Предупреждение — это будущий баг со скидкой.
